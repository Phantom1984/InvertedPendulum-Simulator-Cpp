## 物理参数定义
| 参数名称       | 符号表示   | 说明                     |
|---------------|-----------|-------------------------|
| 小车质量        | M         |                          |
| 摆杆质量        | m         | 长度 l，转动惯量 I=1/3ml² |
| 摆杆角度        | θ         | 逆时针方向为正            |
| 小车位移        | x         | 向右为正方向              |
| 外力输入        | F         | 作用于小车的控制力        |
| 地面摩擦系数    | b         |                          |
